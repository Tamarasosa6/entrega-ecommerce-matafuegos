# Entrega 1: API de Matafuegos

API RESTful para gestionar matafuegos, carritos y usuarios, desarrollada con **Node.js**, **Express** y **MongoDB**.  
Permite registrar usuarios, iniciar sesión con JWT, administrar productos y carritos de compra.

---

## Tecnologías utilizadas

- Node.js  
- Express  
- MongoDB y Mongoose  
- Passport.js (local + JWT)  
- JSON Web Token (JWT)  
- bcrypt (para encriptar contraseñas)  
- Handlebars (opcional para vistas)  

---

## Objetivo

Implementar un CRUD de usuarios con autenticación y autorización, además de endpoints para productos y carritos.  
El objetivo es demostrar manejo seguro de usuarios y sesiones mediante **JWT y Passport**, así como gestión de recursos en una API REST.
